Development Setup: 1. Run npm run dev to start the development server
2. Configure your ChurchTools API settings below
3. The calendar will update automatically as you edit the code

Development Tips

  • Hot Reload: Changes to React components will update automatically
  • TypeScript: Check the terminal for type errors
  • Styles: Edit src/styles/calendar.scss for design changes
  • API: Use browser DevTools Network tab to inspect API calls
  • Components: All components are in src/components/
  • Build: Run npm run build to create production files
